This file shows example output from deployment scripts. Replace all addresses and values with your actual deployment results.

**Export addresses from your deployment:**
```
export FACTORY=<your_factory_address>
export REGISTRY=<your_registry_address>
export NFT=<your_nft_address>
export GOV=<your_governor_address>
```
